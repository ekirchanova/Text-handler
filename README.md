The application allows you to process a file by removing punctuation marks from them and leaving characters longer than the length specified by the user.
First, you need to select the file(s) to process, set the minimum number of words, and mark whether punctuation marks need to be removed.
The next step is to choose how to save the file(s). if there are several files, the name for all of them will be constructed from the specified one.
Next, you can start processing the file, the download progress will be displayed, and you can also cancel the processing.
